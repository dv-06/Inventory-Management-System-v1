# Inventory Control System (v1)

Streamlit and Python were used to create an easy-to-use inventory and order management system. This project uses Excel to show file-based data storage, user authentication, and CRUD operations. 


---

1. Features

    Inventory management, admin authentication, and user registration and login
    Placement and Monitoring of Orders
    Excel-based long-term storage
    Custom CSS with a dark tone and a clean user interface

---

2. Tech Stack

    Excel (.xlsx), Python, Streamlit, and Pandas
